欢迎来到我的个人空间。这是这个博客的第一篇文章，简单聊聊我为什么做它、以及打算写些什么。

## 为什么做这个博客

社交平台的内容是流动的，发完很快就被时间线冲走。我想要一个真正属于自己的角落：内容是我的，格式是我的，节奏也是我的。在这里我可以慢慢写、认真写，不被算法和点赞数牵着走。

## 这里会写什么

- **技术实践**：工作和折腾中踩过的坑、学到的东西
- **思考随笔**：读到、想到、值得记下来的片段
- **项目记录**：做过的小东西和它们背后的故事

## 关于这个站点

这个博客用 [Astro](https://astro.build/) 构建，主题是简洁专注的 AstroPaper。它的特点是：

1. 加载极快——页面默认不带多余的 JavaScript
2. 支持深色 / 浅色模式
3. 自带标签、归档、全文搜索和 RSS 订阅
4. 写作只需要 Markdown

如果你也想搭一个，下面这段代码就是这篇文章的样子——纯文本，干净利落。

```bash
# 写一篇新文章，只需要在 src/content/posts/ 下新建一个 .md 文件
# 然后 git push，几十秒后线上自动更新
```

那么，开始吧。👋
